Detailed Description

Adapter class Responsibility<h1> 1) Session Life Cycle: Physical connection, reconnections, logon and heartbeats 2) grouping inbound data into messages
It is also storing basic ID about the adapter
Inbound data is read in a separate thread which entry point is Adapter::listennerLoop()
